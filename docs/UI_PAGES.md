# RoleFit AI UI Page Catalog

This catalog lists the current frontend pages added to the product shell.

## Public / entry pages

- `/` - Premium SaaS landing page
- `/login` - Login screen UI
- `/career` - Candidate-facing career/application page

## Admin / hiring workspace

- `/dashboard` - AI hiring command center
- `/jobs` - Role pipeline and job blueprint launcher
- `/candidates` - Candidate intelligence directory
- `/applications` - Application pipeline
- `/assessment` - Assessment builder
- `/voice-interview` - AI interview studio shell
- `/fit-report` - Candidate fit report
- `/approvals` - Approval gate
- `/talent-pool` - Talent pool intelligence
- `/analytics` - Recruitment intelligence dashboard
- `/settings` - Tenant settings and provider governance
- `/setup` - Company onboarding setup

## UI direction

- Dark premium SaaS cockpit
- Glass panels
- Gradient highlight system
- Sidebar navigation
- Metric cards
- Status pills
- Candidate evidence reports
- Approval-safe decisioning language

## Production status

These pages are UI-complete shells and can now be connected to real backend endpoints. The backend already includes role, candidate, assessment, interview, integration, and dashboard APIs. Authentication, database migrations, and full form-to-API wiring remain production-hardening tasks.
