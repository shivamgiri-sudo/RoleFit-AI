# RoleFit AI Product Decisions

## Locked decisions

| Decision | Final Choice |
|---|---|
| Product name | RoleFit AI |
| Target customers | All companies |
| First use case | Operations Manager |
| Hiring flow | AI recommendation + human approval |
| Pricing model | Per user |

## Product positioning

RoleFit AI is a broad hiring SaaS platform for companies that want to assess candidates using education, experience, skills, role-specific workplace traits, role alignment, and interview transcript evidence.

The first validated role blueprint is Operations Manager. The product can later expand into all major company roles using the same trait-intelligence framework.

## Decisioning principle

AI can automate workflow, generate evidence, recommend next steps, and prioritize candidates. Final employment actions remain human-approved and audit logged.

## Pricing direction

Per-user pricing will be the primary SaaS model. Candidate assessment volume, AI interview minutes, and advanced analytics can become add-ons later.

## First production scope

1. Operations Manager role blueprint
2. Company onboarding
3. User login and access control
4. Candidate application
5. Trait assessment
6. Deepgram transcript flow
7. OpenAI evaluation flow
8. Role fit report
9. Human approval gate
10. Analytics dashboard
